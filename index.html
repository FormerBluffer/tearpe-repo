Welcome to tearpe.com

I created this website as a personal learning project to learn basic coding skills. I expect it to be a place where I can continue to experiment with new ideas and continue to learn basic web development skills.

Since my focus is on the learning process, I expect that I will publish changes to the website as a work-in-progress and not worry too much about whether everthing works "perfectly". I may include numerous "placeholder" items I expect to complete at a later time. Please be patient if you run into incomplete areas. I hope this does not significantly detract from your experience.

I plan to include various random elements or projects that I have created. These will generally be included in an area I am calling My Experimental Projects. For example, I plan to add pages that reflect my various interests (traveling, outdoor activities, visting the Disney Theme Parks, etc). In addition, many of the sources I am using to learn coding skilles (such as freeCodeCamp) have mini projects that can be completed to learn new skills. I may include some of these projects as well.

I welcome any constructive feedback you care to provide. Also, I would greatly appreciate any suggestions for new content I can add.

Thank you for taking the time to visit my site.
